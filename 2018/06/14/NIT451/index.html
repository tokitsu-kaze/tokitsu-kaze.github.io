等差序列

给你一个长度为n的序列a，你的任务是找到有多少种不同的三元组(i,j,k)，(i,j,k互不相等)，使得a[i]，a[j]，a[k]，为等差序列。

注意：(i,j,k)，(i,k,j)，(j,i,k)，(j,k,i)，(k,i,j)，(k,j,i)视为同一种方案。

Input
多组数据。

对于每组数据，第一行包括一个正整数n(1≤n≤10^5)。

第二行包括n个正整数ai。

Output
对于每组数据，输出一行，表示答案，答案对10^9+7取模。

Sample Input
5
1 2 3 3 3
3
2 1 3
4
1 1 1 1
3
66 666 6666
1
1

Sample Output
4
1
4
0
0

Hint
第一个案例：(1,2,3),(1,2,4),(1,2,5),(3,4,5)。所以答案为4。

第二个案例：(2,1,3)。所以答案为1。

第三个案例：(1,2,3),(1,2,4),(1,3,4),(2,3,4)。所以答案为4。

Source
2017级新生选拔#4

Author
tokitsukaze

题解：FFT计数，与hdu 4609做法类似。注意如果在做FFT时取模，去重时除法要逆元。
